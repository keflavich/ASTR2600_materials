If you would like to use \ipython instead of \IDL, these instructions will get you going.

On the command line, type \ipython to get started.  The very first time you
start up \ipython, you should get this message:

\begin{verbatim}
**********************************************************************
Welcome to IPython. I will try to create a personal configuration directory
where you can customize many aspects of IPython's functionality in:

/home/ginsbura/.ipython
Initializing from configuration: /usr/lib/pymodules/python2.6/IPython/UserConfig

Successful installation!

Please read the sections 'Initial Configuration' and 'Quick Tips' in the
IPython manual (there are both HTML and PDF versions supplied with the
distribution) to make sure that your system environment is properly configured
to take advantage of IPython's features.

Important note: the configuration system has changed! The old system is
still in place, but its setting may be partly overridden by the settings in 
"~/.ipython/ipy_user_conf.py" config file. Please take a look at the file 
if some of the new settings bother you. 


Please press <RETURN> to start IPython.
\end{verbatim}

Go ahead and press \texttt{<RETURN>} as requested.  You'll then see this:

\begin{verbatim}
**********************************************************************
Python 2.6.5 (r265:79063, Apr 16 2010, 13:57:41) 
Type "copyright", "credits" or "license" for more information.

IPython 0.10 -- An enhanced Interactive Python.
?         -> Introduction and overview of IPython's features.
%quickref -> Quick reference.
help      -> Python's own help system.
object?   -> Details about 'object'. ?object also works, ?? prints more.

In [1]: 
\end{verbatim}

You can then quit \ipython by typing \texttt{exit()} or pressing \texttt{control-d}.

Then, from the command line, type:
\begin{verbatim}
cp /home/ginsbura/.ipython/ipy_user_conf.py ~/.ipython/ipy_user_conf.py
\end{verbatim}

This will enable automatic logging every time you start up an \ipython session.  The files
will be stored in \verb|ipython_log_yyyy_mm_dd.py| (e.g., \verb|ipython_log_2012_08_06.py|).
\end{spacing}
